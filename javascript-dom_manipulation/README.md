# JavaScript - DOM Manipulation

This project contains JavaScript scripts that demonstrate DOM manipulation techniques.

## Tasks

- **0-script.js**: Updates the text color of the header element to red
- **1-script.js**: Updates the text color of the header element to red when user clicks
- **2-script.js**: Adds the class red to the header element when user clicks
- **3-script.js**: Toggles the class of the header element between red and green
- **4-script.js**: Adds a li element to a list when user clicks
- **5-script.js**: Updates the text of the header element when user clicks
- **6-script.js**: Fetches and displays a Star Wars character name
- **7-script.js**: Fetches and lists Star Wars movie titles
- **8-script.js**: Fetches and displays a translation of "hello"
